### 属性

|属性名| 说明    | 类型     | 默认值 |
|  ---  |-------|--------|-----|
|  src  | 头像的图片地址 | string | -   |
|  size  | 大小，宽度和高度 | number | 48   |


### CSS 变量

|属性名|说明| 默认值  | 全局变量                       |
|  ---  | ---  |------|----------------------------|
|  --border-radius  | 圆角  | 50%  | --adm-avatar-border-radius |
|  --size  | 大小，宽度和高度  | 48px | --adm-avatar-size |
